# 🎯 Jordan Sneaker Data Analysis Dashboard

A complete sales data analysis project built with Python.

## 📊 Technologies Used
- Python
- pandas
- numpy
- matplotlib
- Jupyter Notebook

## 📈 Analysis Performed
1. Sales Channel Performance
2. Top 10 Most Profitable Shoe Models
3. Monthly Sales Trend
4. Yearly Sales Trend
5. Shoe Condition Impact on Profit
6. Most Valuable Customers
7. Retail vs Resale Price Comparison

## 🗂 Dataset
Air Jordan sneaker market and resale data (2023-2026)
5,000 transactions · 11 columns
